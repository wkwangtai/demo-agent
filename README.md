# Demo Agent

A static demo profile website ready to push to GitHub and deploy on Vercel.

## Preview locally

```bash
npm run dev
```

Then open <http://localhost:3000>.

## Deploy with GitHub and Vercel

1. Create a GitHub repository named `demo-agent`.
2. Push this project to the repository.
3. In Vercel, choose **Add New Project** and import the GitHub repository.
4. Keep the framework preset as **Other** and deploy.

No build command or output directory is required because this is a static site.
